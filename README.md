# ChatSurf

A real-time chat app.

---

## Setup

Run:

```bash
git clone https://github.com/ChatSurfApp/ChatSurf.git
cd ChatSurf
npm start
```

---

## Plugins

#### Finding plugins

- [ChatSurf Plugin Website](https://ChatSurfApp.github.io/plugins)

#### Installing

1. Drag the .js file into the plugins folder.
2. Restart ChatSurf.
